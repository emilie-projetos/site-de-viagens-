#  Site de Viagens

Projeto desenvolvido com HTML e CSS com foco em prática de desenvolvimento web.

## Tecnologias utilizadas:
- HTML
- CSS

## Objetivo:
Criar uma página web com tema de viagens, trabalhando estruturação de conteúdo e design visual.

## Funcionalidades:
- Página estática
- Layout organizado
- Seções informativas sobre destinos

## Como visualizar:
Basta abrir o arquivo index.html no navegador
